# CRO and Content Refresh Guide

## Purpose

Use this when a page gets traffic but does not convert, or when content ranks but needs improvement.

## CRO diagnosis

Check:
- value proposition clarity
- CTA visibility
- trust signals
- proof points
- form friction
- mobile layout
- page speed
- message match
- objection handling
- internal links to case studies or related pages

## High priority CRO fixes

High priority when:
- page has high traffic but low leads
- page targets commercial intent
- page ranks for valuable queries
- page lacks clear CTA or trust proof
- mobile users struggle to convert

## Content refresh rules

Do:
- preserve working search intent
- improve clarity and usefulness
- add proof, FAQs, comparisons, process, pricing guidance, and case studies
- improve internal links
- update outdated sections
- strengthen titles and meta only when evidence supports it

Do not:
- rewrite ranking pages blindly
- remove useful content without checking query performance
- stuff keywords
- change URLs unnecessarily
- copy competitor content

## Ranking but not converting

If a page ranks but does not convert:
1. Check query intent.
2. Check whether the offer matches the intent.
3. Improve above-the-fold value proposition.
4. Add trust/proof near the first CTA.
5. Reduce form friction.
6. Add relevant FAQs.
7. Track form_submit, phone_click, email_click, and quote_request.

## Content refresh evidence needed

Ask for:
- URL
- target query
- GSC query/page data
- GA4 conversion data
- current title/meta/H1
- competitor examples
- recent changes
- business goal
